QuestStudy

Gamified Stress-Free EdTech Ecosystem

The Big Idea

Students target core academic anxiety everyday. QuestStudy enters as a hybrid framework that redesigns tedious syllabus trackers into gamified daily quests. The objective is simple: reduce academic pressure through psychological positive reinforcement.

Core Systems Built

🌐 Dual Mechanics (Online + Offline)

Online mode syncs quiz battles and digital micro-tasks, while the Offline module implements automated Pomodoro intervals to track real-desk library study sessions without screen distractions.

🪙 Task Economy (Game Coins)

Completing dynamic practice sets, mock exams, or consistent study targets rewards users with proprietary profile coins. These coins act as tokens to customize dashboard tiers or trade for real student perks.

🧠 Stress-Free Interface Layout

Designed natively on dark zen interfaces to minimize biological cognitive overload. Heavy performance pressure gauges are replaced with non-punitive character progression mechanics.
